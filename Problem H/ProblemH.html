Problem H - Janny's Beautiful Ring

Description

After fixing Janny's photos and gazing at them, Jaffy decided to get Janny a beautiful present. Jaffy knows that Janny loves jewelry in general and rings in particular. Jaffy does not have much money, which is typical for college students. Jaffy decides he can create a ring for Janny and print it using UMW's free 3D printer. The programming contest must settle for 2D printing. Print Janny's beautiful ring.
